# Project: LectureLive

## Language
All communication and code comments should be in Chinese (中文).

## Tech Stack
- Next.js 15 + React 18 + TypeScript
- Tailwind CSS 3.4 (custom theme: cream, charcoal, rust color palette)
- Zustand for state management
- Socket.IO for real-time communication
- Prisma ORM

## Key Conventions
- Components use 'use client' directive
- Stores are in `/src/stores/`
- Types are in `/src/types/`
- API routes follow Next.js App Router convention
